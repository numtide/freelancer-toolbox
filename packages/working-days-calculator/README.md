# working-days-calculator

Count unique working days from a [Harvest](https://www.getharvest.com) time report CSV export.

## Getting the CSV

Go to [Harvest Reports](https://numtide.harvestapp.com/reports?kind=year), filter by year, and export the CSV.

## Usage

```console
working-days-calculator report.csv
```

Output:

```
Working days: 42 from 2024-01-02 to 2024-12-31
```
