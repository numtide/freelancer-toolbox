## Contributing

We welcome contributions for any country! To add or improve a country guide:

1. Create or edit `docs/countries/{country}.md`
2. Include sections for:
   - Tax registration process
   - Health insurance options
   - Recommended accounting software
   - Any country-specific legal considerations
   - Links to official resources
3. For country-specific documents (PDFs, forms), create a subdirectory: `docs/countries/{country}/`
